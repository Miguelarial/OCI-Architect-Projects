# Secure Cloud Infrastructure Deployment on OCI

## Overview
This project demonstrates the design and deployment of a secure cloud infrastructure using Oracle Cloud Infrastructure (OCI). The primary goal is to showcase the ability to set up and secure cloud resources effectively.

## Table of Contents
- Setup Guide 
- Security Best Practices
- Monitoring Guide


## Features
- Virtual Cloud Network (VCN) setup with proper subnetting and security lists.
- Secure compute instance deployment with essential configurations.
- Create a block volume and attach the block volume to the instance.
- Monitoring setup with alarms for resource health monitoring.
